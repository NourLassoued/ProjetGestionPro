# 🚀 ProjetGestionPro - Application Web de Gestion de Projets

**Stage de développement Full-Stack** | Billcom Consulting Tunisie (Juin - Août 2024)

Application web collaborative de gestion de projets et d'équipes, permettant le suivi en temps réel des tâches et l'organisation agile du travail.

## 💼 Impact Business
- ✅ Collaboration centralisée pour équipes distribuées
- ✅ Suivi en temps réel de l'avancement des projets
- ✅ Optimisation de la productivité des équipes
- ✅ Sécurité renforcée avec authentification JWT

## 🛠️ Stack Technique Full-Stack
**Backend:** Spring Boot 3.2 • Spring Security • JWT • RESTful API • MySQL • JPA/Hibernate  
**Frontend:** Angular 17 • TypeScript • RxJS • Angular Material  
**DevOps:** Docker • Jenkins CI/CD • SonarQube • Kubernetes • ArgoCD • GitOps  
**Testing:** JUnit 5 • Mockito • JaCoCo • Tests unitaires & intégration (238 tests)

## 🎯 Réalisations Clés

### 🔐 Sécurité & Authentification
- Architecture sécurisée avec **Spring Security 6** et tokens **JWT**
- Système d'authentification avec refresh tokens
- Gestion des rôles et permissions (RBAC)
- Protection CSRF et validation des entrées
- Endpoints publics/privés avec contrôle d'accès

### 🧪 Qualité & Tests
- **238 tests automatisés** (unitaires + intégration)

### 🚀 DevOps & CI/CD
- Pipeline automatisé complet : **Jenkins** → **SonarQube** → **Docker** → **Kubernetes**

- Containerisation avec **Docker** et orchestration **Kubernetes**
- Déploiement GitOps avec **ArgoCD**

### 💻 Développement
