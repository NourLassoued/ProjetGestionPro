# ProjetGestionPro
Projet réalisé de juin 2024 à août 2024 chez Billcom Consulting Tunisie.

Cette application web permet de gérer des projets collaboratifs en facilitant la communication et l’organisation des équipes.

🔧 Fonctionnalités principales :

Gestion des projets
Gestion des tâches (création, modification, suivi)
Collaboration entre utilisateurs
Organisation du travail en équipe

💻 Technologies utilisées :

Backend : Spring Boot (Java)
Frontend : Angular
Base de données : MySQL
